## 📌 Project Overview

This project analyzes a retail **Superstore sales dataset** to uncover sales trends, customer behavior, product performance, and regional performance. The goal is to transform raw transactional data into clear, actionable business insights that support data-driven decision-making — using Excel as a complete analytics tool, from raw data to a polished interactive dashboard.

---

## 🎯 Business Objective

The primary objective of this project is to analyze retail sales data and answer key business questions, including:

| # | Business Question |
|---|---|
| 1 | Which region generates the highest and lowest sales? |
| 2 | Which product categories contribute the most revenue? |
| 3 | Which customers generate the highest sales? |
| 4 | Which products have the highest demand? |
| 5 | What are the monthly sales trends? |
| 6 | How can the company improve overall business performance? |

## 🧹 Data Cleaning Performed

- Created a separate copy of the raw dataset to preserve the original.
- Standardized date formats across the dataset.
- Investigated missing values and their impact on analysis.
- Identified missing postal codes (Burlington, Vermont).
- Preserved data integrity by avoiding unsupported assumptions.
- Documented all data quality issues found during cleaning.
- Validated dataset structure before beginning analysis.

## ⚠️ Data Quality Observations

During data validation, several issues were identified and documented:

- Mixed date formats
- Missing postal codes
- Inconsistent text formatting
- Shipping date anomalies

These issues were investigated and addressed before performing business analysis to ensure the final insights were reliable and trustworthy.
